Sure, here's a simple example of a Flask app with a navigation bar/sidebar for navigating to different pages:

python
Copy code
from flask import Flask, render_template

app = Flask(__name__)

# Define routes for different pages
@app.route('/')
def home():
    return render_template('home.html')

@app.route('/about')
def about():
    return render_template('about.html')

@app.route('/contact')
def contact():
    return render_template('contact.html')

if __name__ == '__main__':
    app.run(debug=True)
Now, let's create templates for these pages. Inside a folder named templates, create three HTML files: home.html, about.html, and contact.html.

home.html:

html
Copy code
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Home</title>
</head>
<body>
    <nav>
        <ul>
            <li><a href="/">Home</a></li>
            <li><a href="/about">About</a></li>
            <li><a href="/contact">Contact</a></li>
        </ul>
    </nav>
    <h1>Welcome to the Home Page</h1>
</body>
</html>
